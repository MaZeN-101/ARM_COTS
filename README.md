# ARM_COTS
Implementing most of the peripheral drivers of STM32f10 Microcontroller based on ARM (MCAL Layer):
1. RCC
2. GPIO
3. PORT
4. NVIC
5. EXTI
6. AFIO
7. SysTick
8. SPI

Interfacing with HW devices (HAL Layer):
1. LED & RGB LED
2. Seven Segments
3. SWITCH 
4. LED MATRIX
5. NEC IR Communication
6. Serial to Parallel Communication
7. TFT LCD

(Service Layer) :
1. RTOS

(Library Layer) :
1. Standard Types
2. Bit Math
